1) What is a Computer?
A computer is an electronic device that processes data to perform tasks and execute instructions.

2) What is RAM?
RAM is a temporary memory in a computer that stores data needed to run applications and open files.

3) Where is data stored in a computer?
Data is stored in a computer on storage devices like hard drives, SSDs, or removable media.

4) What is that input device used to type text and numbers on a document in the computer system?
The input device used to type text and numbers is the keyboard.

5) What are the output devices?
Output devices are used to display or convey results to the user.

6) Which is the input device that allows a user to move the cursor or pointer on the screen?
The input device used to move the cursor or pointer is the mouse.

7) Which language is directly understood by the computer without a translation program?
Machine language is directly understood by a computer without a translation program.

8) What are Input devices?
Input devices are hardware tools used to send data to the computer.
